## 🚀 Como Rodar o Projeto

### 1️⃣ Subir Banco de Dados
docker-compose up -d

### 2️⃣ Instalar Dependências
npm install

### 3️⃣ Rodar Migration
npx prisma migrate dev --name init

### 4️⃣ Gerar Prisma Client
npx prisma generate

### 5️⃣ Rodar Seed
npm run seed

### 6️⃣ Iniciar Projeto
npm run dev
